quotient group

a quotient group is a subgroup where you mask out multiples:

\begin{equation} \mathbb{Z}} / 3 \mathbb{Z} \end{equation}

this is essentially the same thing as \(\mod 3\).

However, each value in this new group is a group itself, a set of numbers; for instance, in \(\mathbb{Z} / 3\mathbb{Z}\), \(0\) is actually the set \(\{\dots -6, -3, 0, 3, 6, \dots\}\); we call this a subgroup.

Other elements in the quotient space (“1”, a.k.a. \(\{ \dots, -2, 1, 4, 7 \dots \}\), or “2”, a.k.a. \(\{\dots, -1, 2, 5, 8 \dots \}\)) are called “cosets” of \(3 \mathbb{Z}\). You will notice they are not a subgroups.
