range

Last edited: November 11, 2022 The range (image, column space) is the set that some function \(T\) maps to. constituents some \(T: V\to W\) requirements The range is just the space the map maps to: \begin{equation} range\ T = \{Tv: v \in V\} \end{equation} additional information range is a subspace of the codomain This result is hopefully not super surprising. zero \begin{equation} T0 = 0 \end{equation} as linear maps take \(0\) to \(0\), so \(0\) is definitely in the range.
