You can show it to yourself that multiples of any number in the group is a subgroup of that group. For instance: \(3 \mathbb{Z}\), the set \(\{\dots -6, -3, 0, 3, 6, \dots\}\) is a subgroup actual quotient groups We can use the subgroup above to mask out a group.
