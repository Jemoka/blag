Division!

constituents

  • integers \(a, b\)

requirements

\(b\) is said to divide \(a\) if there is some integer \(q\) such that:

\begin{equation} a = qb \end{equation}

This is written as: \(b|a\).

additional information

greatest common divisor

See also greatest common divisor

division algorithm

For \(a,b \in \mathbb{Z}\), with \(b>0\), there is some unique \(q, r\) such that \(a = qb+r\), where \(0 \leq r < b\).

We call \(q\) the quotient and \(r\) the remainder.
